# 测试数据样例

本文件展示了语料文件的格式示例。

## Excel/CSV格式要求

### 必需字段

- `content`: 语料内容（必需）

### 可选字段

- `type`: 语料类型（"请求"或"反馈"）
- `time`: 时间戳

## 示例数据

| content | type |
|---------|------|
| 希望PPT模板适合老师上课使用，简洁一点 | 请求 |
| 能否增加教学案例展示的模块？ | 请求 |
| 老师反馈说操作太复杂，不适合课堂演示 | 反馈 |
| 教学PPT的页数能控制在10-15页吗？ | 请求 |
| 页面太多，老师上课时间不够用 | 反馈 |
| 希望有更多适合教学场景的PPT风格 | 请求 |
| 学生反馈PPT内容很好，但字体太小 | 反馈 |
| 需要权限管理功能，方便多个老师协作 | 请求 |

## 创建测试文件

你可以：
1. 使用Excel创建上述格式的文件
2. 使用CSV格式保存（UTF-8编码）
3. 确保至少包含`content`列

## 使用测试数据

```bash
# 使用命令行测试
python src/main.py test_data.xlsx --dimensions "老师,教学" --type both

# 使用GUI测试
python src/gui.py
# 然后在界面中选择文件
```
